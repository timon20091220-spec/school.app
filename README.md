# 배문고 통합 학교 앱 · 연봉제 시제품

배문고 전체 앱 안에 `연봉제` 축제 모드를 포함한 모바일 우선 PWA 시제품입니다.

## 로그인 정보
- 학생 로그인: 아무 학번 + 이름
- 관리자 로그인: `admin` / `2026`

## 포함 화면
- 학교 앱 홈
- 층별 지도 및 경로 안내 시뮬레이션
- 연봉제 홈
- 체험 원거리 예약 시뮬레이션
- 먹거리 장터 메뉴판
- 버스킹·주요 행사 알림/진동 시뮬레이션
- 연결 프로그램 콘셉트 화면
- 내 정보 화면

## GitHub Pages 업로드
압축을 푼 뒤 폴더 안의 파일을 저장소 최상단에 업로드하세요.

필수 구조:
- index.html
- styles.css
- app.js
- manifest.json
- sw.js
- icons/

그 후 Settings → Pages → Deploy from a branch → main → /(root)를 선택하세요.

## 현재 제한
- 실제 예약 서버는 연결되지 않았습니다.
- 지도와 길찾기는 시뮬레이션입니다.
- 푸시 알림은 권한 요청과 진동 미리보기만 포함됩니다.
- 예약은 현재 기기의 localStorage에만 저장됩니다.


## 관리자 기능
- 예약 현황 확인
- 예약 초기화
- 공지 수 카운트 시뮬레이션
- 행사 알림 진동 테스트
- 연봉제 모드 ON/OFF
- 예약 명단 CSV 다운로드


## 공지 관리 기능
관리자 로그인 후 학교 공지를 직접 등록할 수 있습니다.

- 관리자 계정: `admin`
- 관리자 비밀번호: `2026`
- 분류: 일반 / 시험 / 행사 / 긴급
- 대상: 전체 / 1학년 / 2학년 / 3학년
- 관련 날짜 기반 D-Day 자동 계산
- 중요 공지 상단 고정
- 공지 삭제 및 고정 해제
- 알림 권한이 허용된 기기에서 등록 알림 테스트

현재 공지와 로그인은 기기 내부 저장소 방식입니다. Firebase 연결 전에는 다른 기기와 공유되지 않습니다.
