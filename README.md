# 배문고 학교 앱 v5

관리자 로그인: `admin / 2026`

## 주요 기능
- 관리자 로그인 시 앱 재실행과 홈 버튼 모두 관리자 홈으로 이동
- 오늘의 배문 직접 수정
- 여러 축제 생성, 공개/숨김, 대표 축제 설정
- 축제별 체험 부스, 메뉴, 행사 일정 관리
- 부스 예약 시간을 비우면 즉시 예약, 입력하면 시간 선택 예약
- 공지 요약/상세/사진 및 수정·삭제
- 사진 클릭 전체 화면 보기
- 행사 알림 작성 및 알림함 누적, 알림 켜기/끄기

현재 데이터는 같은 기기의 브라우저 저장소에만 저장됩니다. 실제 전체 학생 공유는 Firebase 연결이 필요합니다.
